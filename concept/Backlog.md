Содержимое этого файла не влияет на подсчёт `diff = 0`.

как новую `F-NN`._

## Идеи

- [ ] **In-app создание/закрытие задач прямо в городе** — клик на пустую плитку
      квартала → ввести задачу и сразу её закрыть (или ввести план и галочкой
      отметить выполнение позже). По сути встроенный мини-task-manager поверх tasks.jsonl.
      Идея: писать задачи прямо в город, без отдельного редактора `.md`.
      _Добавлено 2026-05-22_
- [ ] **Reference-агрегатор markdown → tasks.jsonl** — отдельная утилита, парсит
      `.md` файлы с фронтматтером `project:` и `[x]` чекбоксы, дописывает в
      `tasks.jsonl`. Это **внешний инструмент**, не часть игры (контракт — только
      `tasks.jsonl`). Полезен пользователям, которые ведут задачи в `.md`.
      _Добавлено 2026-05-21_
- [ ] **Дедупликация по `task_id`** — если строка с тем же `task_id` уже была
      обработана, игра её игнорирует. Защита от двойной записи кроном.
      _Добавлено 2026-05-21_
- [ ] **Переименование проекта** — операция миграции «слить квартал A с кварталом B»
      или «переименовать project='foo' в project='bar'». _Добавлено 2026-05-21_
- [ ] **Экспорт таймлапса** — видео-ролик роста города за выбранный период
      (использует event log, реплеит ускоренно, рендерит в mp4). _Добавлено 2026-05-21_
- [ ] **День/ночь с динамическим освещением** — суточный цикл, ночью окна светятся,
      фонари у дорог, патрули с факелами. _Добавлено 2026-05-21_
- [ ] **Сезонные события и праздники** — при достижении milestone (закрыт большой проект,
      серия активных недель) — фестиваль, фейерверк, новый монумент. _Добавлено 2026-05-21_
- [ ] **Звук** — эмбиент (ветер, птицы, рынок), эффекты (молот, колокол).
      _Добавлено 2026-05-21_
- [ ] **macOS виджет с превью города** — в Notification Center показывает миниатюру и
      счётчик населения. _Добавлено 2026-05-21_
- [ ] **Уведомления при decay** — нативный push «проект X начал ветшать, 5 дней без
      активности». _Добавлено 2026-05-21_
- [ ] **Несколько городов** — разные карты по сферам жизни (работа, личное, учёба).
      Переключение между ними. _Добавлено 2026-05-21_
- [ ] **Достижения / события-награды** — закрыл 100 задач → монумент; серия 30 дней →
      храм. Визуальная и геймификационная награда. _Добавлено 2026-05-21_
- [ ] **Караваны между кварталами** — спрайты-торговцы ходят между связанными
      проектами, визуализируют взаимосвязь. _Добавлено 2026-05-21_
- [ ] **Интеграция с Reminders / Things / Linear** — прямые адаптеры к этим источникам,
      без обязательного использования `.md` файлов. _Добавлено 2026-05-21_
- [ ] **Mini-Map / обзор всей карты** — отдельное окно с zoom-out видом всего города.
      _Добавлено 2026-05-21_
- [ ] **Тема дня (light/dark стиль ассетов)** — синхронизация с системной темой macOS.
      _Добавлено 2026-05-21_
- [ ] **Dedup русских названий `UnitKind` → одна точка истины** — сейчас таблица «Лачуга/Дом/Вилла/…»
      существует в 3 местах: `UnitKind.label` (Data/CityState.swift, добавлено в TASK-024) и
      приватные `russianKind` в `InspectorPanel.swift` и `InspectorOverlayCard.swift`. Заменить
      приватные функции на `unit.kind.label` — устранить дублирование. _Источник: TASK-024
      lead-разбор (out-of-scope), 2026-05-23._
- [ ] **Partial-snapshot robustness** — при decode ошибки в одном юните (например,
      пользователь отредактировал `state.json` вручную и сломал один `kind`) сейчас весь
      snapshot инвалидируется и движок идёт на full replay. Более мягкий вариант: пропускать
      битый юнит, загружать остальные. Требует кастомного `init(from:)` для `CityState`
      с пер-юнитным `try?` + знания фактического JSON-layout словаря `[UUID: UnitState]`.
      _Источник: TASK-037 lead-разбор (out-of-scope), 2026-05-23._
